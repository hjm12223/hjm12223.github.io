모나리(모여서 나누는 우리) 2025 04 ~ 2025 05

개요: 사교육비 부담 완화를 위한 공동구매 개념을 도입한 C2C 수업 중개 플랫폼입니다.

GitHub: https://github.com/team-monari

기술 스택: Spring Boot 3.4.4, Spring Batch, Java 17, AWS(EC2, RDS, S3), Docker, MySQL, Nginx, QueryDSL, JPA, GitHub Actions, 서울시 시설대관 공공서비스 예약정보 API

주요 역할 및 기여:

  • 수업 및 등록 도메인 API 개발을 주도하여 플랫폼의 핵심 기능을 구현했습니다.
  • 서울시 공공 API를 활용, 학원/스터디카페 정보를 수집하고 정합성을 100% 유지하는 배치 시스템을 구축했습니다.
  • 프론트엔드와 백엔드의 CI/CD 파이프라인을 구축하여 개발 및 배포 효율성을 극대화했습니다.

성과 및 문제 해결:

  • N+1 쿼리 최적화를 통해 수업 목록 API 응답 속도를 P99 기준 78% 단축 (850ms → 190ms)하고, 서버 비용을 30% 절감했습니다.
  • 배치 시스템 개선으로 공공 데이터 정합성을 100% 보장하며 stale data 발생률을 0%로 만들었습니다.
  • 페이징 쿼리 최적화로 응답 시간을 4,345ms → 23ms (99.5% 단축)하였고, TPS를 180배 향상시켰습니다.
  • Nginx 기반 HTTPS를 도입하여 보안을 강화하고 인프라 비용을 40% 절감했습니다.

Koreano(코레아노) 2024 04 ~ 2024 06

성과 및 문제 해결:

  • Outbox Pattern 도입으로 주문-재고 데이터의 정합성을 100% 보장하고 Race Condition을 제거했습니다.
  • Jacoco 기반 테스트 자동화를 구축하여 PR/Push 시 자동 커버리지 측정을 통해 개발 효율성을 향상했습니다.
